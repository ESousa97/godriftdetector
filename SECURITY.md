# Security Policy

A segurança deste agente e da infraestrutura que ele monitora é a nossa maior prioridade.

## Supported Versions

Por favor, reporte vulnerabilidades apenas nas versões atualmente suportadas do projeto.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Se você encontrar uma vulnerabilidade de segurança, por favor **não** abra uma Issue pública.

Em vez disso, envie um e-mail confidencial para: [xfuuuuuuuuury2018@gmail.com](mailto:xfuuuuuuuuury2018@gmail.com).

Ao reportar, por favor forneça:
- A versão do `godriftdetector` que você está utilizando.
- Detalhes precisos sobre como reproduzir a falha.
- Se possível, um script ou payload demonstrando o problema.

## SLA de Resposta

Analisaremos a denúncia em até **72 horas** e manteremos o relator atualizado sobre o processo de correção e a data de lançamento da patch release. Agradecemos a prática de *responsible disclosure*.
